Typography

Fonts/typography, when used consistently, unifies messaging and creates familiarity. The following guidelines will help ensure maximum legibility and reinforcement of the SJI brand.

Benton Sans

Benton Sans is our primary typeface. Title and sentence case are preferred. Only use all caps in small instances such as client name. The default weight to use is Book | 500.

Headlines should be bold, all lowercaps, solid black #000.

Open Sans

Type Scale

We follow a 6px baseline grid for achieving a vertical rhythm on all block-level elements in the app.

Paragraphs

The default body text is 18px and color is #333. For optimum reading length, the paragraph width is capped at 635px.
